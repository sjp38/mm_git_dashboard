Archives list and review status of patches in mm.git, per tree and subsystem.

The files hiearchy is like below:

- `patches/` directory contains the patch files for commits in mm.git, as of the
  time of the update of this repo.  Files under the directory having `.series`
  suffix are listing the baseline and list of patches on the branch of the
  name.
- `summary/` directory contains the list and review status of patches in mm.git,
  per tree.
  - `summary` file shows summary of the status, with only counts.
  - `full_commits_list` file shows the full list of the commits per branch,
    with review status.
  - `changes_from_last_update` file shows the changes on the list and review
    status of the patches per tree, compared to the last update of this repo.
  - `commits_info.json` is a `json`-format file containing all metadata for the
    list and review status of the patches.
  - `subsystem/` directory contains `summary`, `full_commits_list` and
    `changes_from_last_update` files for each subsystem.

The contents are automatically generated by
[mm_tree_snapshot.sh](https://github.com/sjp38/lazybox/blob/master/linux_hack/mm_tree_snapshot.sh).
